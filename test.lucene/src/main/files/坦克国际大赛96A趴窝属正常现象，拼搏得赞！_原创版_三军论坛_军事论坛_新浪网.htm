         一、其他国家参赛选手，已在俄国场地提前半个月适应性训练，对赛事规则，竞赛要领非常明白，对比赛场地已非常熟悉。而中国队是在赛前3天才到达俄罗斯，适应场地时间短，对比赛要求规则的熟练度也不及他国选手，这些因素必然影响到中国选手的发挥。         二、俄国选手对自身坦克性能，对场地及赛场难点之处，必做到了预先准备，所以俄罗选手有主场之利。这好比，演习情境，一个是告知了科目，一个是随机科目，难度自然不一样。         中国选手驾96A坦克在异国与熟手竞技，感到气势不输，体现了中国军队威武之师、文明之师的形象。相信后阶段的比赛，中国队的成绩会有所提高，但要想超过东道主俄国队，可能比较困难。因为，比赛规则是东道主制订的，设置的比赛科目也是东道主最能发挥擅长的，地形也是东道主再熟悉不过的，所以强龙难按地头蛇。     派96A坦克去俄国比赛，不管最后比赛结果名次如何，相信必能提高96A坦克的知名度。这次比赛能让其他国家选手近距离感知96A坦克的性能，必能促进96A坦克的出口竞争力。同时通过竞赛，增进了中国与各国军队的友谊，增加了互相理解，降低了军事冲突的风险，对促进亚态及世界和平有莫大的意义。 [ 本帖最后由 底层人民站起来 于 2014-8-6 11:45 编辑 ]    批注： 收藏评分分享
