09:49  中国新闻网 我有话说 收藏本文 中新网8月10日电 据外媒报道，加拿大安大略省警方9日表示，3名美国人在安大略省西北方的坠机事故中身亡。 报道称，当地时间8日，一架小型飞机在安大略省坠机。救援小组下午在安大略省肯诺拉南方的查派湖找到失事的赛斯纳182型飞机。 警方说，现场有2具遗体，第3人送医不治。 加拿大有关单位认定，3名死者都来自美国明尼苏达州大急流城。
