08:15  环球时报  我有话说 收藏本文 资料图：美国海军弗吉尼亚级核攻击潜艇 资料图：俄军伊尔-38反潜机 俄罗斯《晨报》10日报道称，俄罗斯海军总参谋部高级官员9日表示，俄罗斯北方舰队的反潜力量发现并从本国边境水域“赶走”美国一艘核潜艇。他说：“8月7日，北方舰队值班部队在巴伦支海海域发现一艘外国潜艇，据推测，是美国海军‘弗吉尼亚’级核潜艇。为此，海军向指定区域派出反潜攻击舰群和1架伊尔-38反潜侦察机进行搜索和追踪。”整个“驱赶”行动持续约27分钟，美国潜艇被迫离开这一海域。 据报道，俄罗斯北方舰队前司令维亚切斯拉夫・波波夫表示，从苏联时期起，美国潜艇追踪俄罗斯军舰活动的行动就一直没有停止过。一段时间，他们的活动频率有所降低。而现在，美国又加强在俄罗斯海岸地区跟踪俄军舰和潜艇的活动。美国拒绝俄美领导人签署的避免水下发生事故的协议。因为美国不愿受到这一条约的约束，这样美方可以随意接近俄罗斯沿海地区活动。来自俄共的国家杜马国际事务委员会副主席卡拉什尼科夫表示，在俄海域发现美国潜艇是一件十分正常的事情。但在当前俄美正发动信息战的背景下，将引起国际社会的高度关注。 俄美的这次“猫捉老鼠”好戏也确实引起第三国的关注。据日本“东京快报”网站10日报道，美国最新锐攻击核潜艇在北极圈的俄罗斯领海附近被发现，并被俄军部队逼退。作为静音性能和躲避声呐追踪性能卓越的美国最新锐潜艇，如此轻易地被俄罗斯反潜搜索部队发现，暴露出美军攻击核潜艇在战争中的弱点。有情报称，欧洲潜艇专家认为，俄罗斯之所以如此轻易地发现美国潜艇，是因为目前正在俄罗斯避难的前美国情报人员斯诺登向俄泄露了美军的“最高机密”。但此说法目前尚未得到证实。 由于在俄罗斯家门口侦察俄核潜艇的动向，北约海军还曾与俄罗斯海军发生过多次严重碰撞事故。1992年2月，美国海军“洛杉矶”级“巴吞鲁日”号核潜艇在科拉半岛军港入口处跟踪俄罗斯北方舰队的“塞拉”级核潜艇K-276“螃蟹”号。俄潜艇发觉后，立即采取迎头对撞的战术动作，向美国潜艇冲了过来。俄潜艇的指挥台围壳撞上美国潜艇的艇身，其围壳几乎被彻底撞毁，美方潜艇耐压艇体则遭受致命撞击，被迫在3年后退役，成为第一艘退役的“洛杉矶”级潜艇。1986年英国一艘核潜艇曾与北方舰队的“台风”级战略核潜艇发生相撞。
